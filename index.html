HUGO MINIS

Keep it simple, and stupid.

Minis is a Hugo theme written by Wejan, it’s modified from Yihui’s XMin theme, most of contents and styles are from xmin. The main motivation for writing this theme was to provide a simple and minimal theme for myself. Here is a website as a demo.

Like XMin, it is a minimal, simple and fully functional theme. It supports pages (including the home page), blog posts, a navigation menu, categories, tags, and RSS.

All pages not under the root directory of the website are listed below. You can also visit the list page of a single section, e.g., posts. See the About page for the usage of this theme.
